### 4. **script.js** - ملف JavaScript الرئيسي

**الوظيفة:** جميع الوظائف التفاعلية للصفحة الرئيسية

**المحتوى:**

#### **أ. إدارة اللغة:**
```javascript
var currentLang = 'ar';  // اللغة الحالية
```
- **initializeLanguage():** تهيئة اللغة
- **toggleLanguage():** تبديل اللغة
- **setLanguage():** تعيين اللغة وتحديث جميع العناصر

#### **ب. بيانات المنتجات:**
```javascript
var products = [
    {
        id: 1,
        name: { ar: 'فلتر زيت محرك', en: 'Engine Oil Filter' },
        description: { ar: '...', en: '...' },
        category: 'engine',
        price: { ar: '150 ج.م', en: '150 EGP' },
        icon: '🔧'
    },
    // ... 18 منتج
];
```
- **18 منتج** مقسمة على 5 فئات:
  - **Engine (محرك):** 6 منتجات
  - **Brakes (فرامل):** 3 منتجات
  - **Suspension (تعليق):** 3 منتجات
  - **Electrical (كهرباء):** 3 منتجات
  - **Body (هيكل):** 3 منتجات

#### **ج. عرض المنتجات:**
- **initializeProducts():** تهيئة عرض المنتجات
- **renderProducts():** عرض المنتجات حسب الفئة
- **initializeFilters():** تهيئة الفلاتر

#### **د. القائمة المتحركة (Mobile Menu):**
- **initializeMobileMenu():** تهيئة قائمة الموبايل
- فتح/إغلاق القائمة عند الضغط على الزر

#### **هـ. السلايدر (Hero Slider):**
- **initializeHeroSlider():** تهيئة السلايدر
- **changeSlide():** تغيير الشريحة
- **goToSlide():** الانتقال لشريحة معينة
- **startAutoSlide():** بدء التمرير التلقائي (كل 5 ثواني)
- **stopAutoSlide():** إيقاف التمرير التلقائي

#### **و. Smooth Scroll:**
- **initializeSmoothScroll():** تمرير سلس للروابط الداخلية

**المميزات:**
- ✅ تخزين اللغة في localStorage
- ✅ تحديث ديناميكي للمحتوى
- ✅ سلايدر تلقائي مع إمكانية التحكم

## 🛠️ كيفية التعديل

### **إضافة منتج جديد:**

1. افتح `script.js`
2. أضف منتج جديد في مصفوفة `products`:
```javascript
{
    id: 19,
    name: { ar: 'اسم المنتج', en: 'Product Name' },
    description: { ar: 'الوصف', en: 'Description' },
    category: 'engine', // أو 'brakes', 'suspension', 'electrical', 'body'
    price: { ar: 'السعر ج.م', en: 'Price EGP' },
    icon: '🔧' // أي emoji
}
```

### **تغيير الألوان:**

1. افتح `styles.css`
2. عدّل المتغيرات في `:root`:
```css
:root {
    --primary-color: #1a56db;  /* غيّر هذا */
    --secondary-color: #0f3a8a; /* غيّر هذا */
    --accent-color: #f59e0b;    /* غيّر هذا */
}
```

### **تغيير معلومات الاتصال:**

1. افتح `index.html`
2. ابحث عن قسم "Contact Section"
3. عدّل المعلومات
